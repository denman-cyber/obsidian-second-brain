# Inbox legacy

Denne mappe er nu legacy.

Ny indbakke er:

```text
raw/
```

Gamle noter her kan flyttes til `raw/` eller `90_Archive/`.

Eksempler:
- hurtige noter fra Telegram
- artikler eller links der skal bearbejdes
- ideer der ikke har en fast plads endnu
- rå tanker, som senere kan blive opgaver, projekter eller vidensnoter
