# thellufsenfoto Hermes

Dette område er scoped til thellufsenfoto og kan senere bruges af Hermes/VPS-agenten.

## Mapper

- `raw`: nye beskeder, briefs og input.
- `processed`: behandlede input.
- `projects`: aktive fotoopgaver og shoots.
- `clients`: kundekort.

## Regel

Eksterne agenter må kun arbejde i denne mappe og dens undermapper.
